### Приложение CodeScan для терминала сбора данных M3UL20X

#### Задачи приложения
1. Отслеживание события `считывание штрихкода`
2. Регистрация считывания
3. Определения явлфется ли считанный штрих-код кодом линии\робота или кодом короба
4. Отправка запроса на запись в MSSQL предприятия
5. При удачной записи запросить у MSSQL последние 20 строк и вывести их в историю
6. Обрабатывать возникающие ошибки
